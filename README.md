
# Модуль `Programm.cs`

## Описание

Модуль `Programm.cs` предназначен для запуска заданий по нажатию соответствующих клавиш на клавиатуре. Каждое задание связано с методом `Execute()` в классах, названия которых соответствуют номерам заданий. Это позволяет пользователю быстро и удобно выполнять задания, используя простой интерфейс командной строки.

## Установка

Для использования модуля убедитесь, что у вас установлен .NET 8.0 и выше. Скомпилируйте проект с помощью следующей команды:

```bash
dotnet build
```

## Использование

1. Запустите приложение:

   ```bash
   dotnet run
   ```

2. После запуска вы увидите инструкции на экране:

   ```
   | Система Запуска Заданий |
   | Для продолжения выберите номер задания и нажмите соответствующую клавишу |
   | Пример: клавиша 1 - Задание 1 | Для выхода 'Esc' |
   ```

3. Нажмите клавишу от 1 до 9 для выполнения соответствующего задания. Например, нажатие клавиши `1` выполнит `Execute()` метод класса `Task1`.

4. Чтобы выйти из программы, нажмите клавишу `Esc`.

## Структура кода

- **`Main`**: Основной метод, который контролирует цикл выполнения программы и обрабатывает нажатия клавиш.
- **`DisplayInstructions`**: Метод, который выводит инструкции по использованию программы.
- **`ExecuteTask`**: Метод, который использует рефлексию для поиска и вызова метода `Execute()` соответствующего класса задания. Если класс или метод не найдены, выводится сообщение об ошибке.

## Примечания

- Убедитесь, что классы заданий (например, `Task1`, `Task2` и т.д.) реализуют метод `Execute()` для корректной работы.
- Метод `Execute()` в каждом классе задания должен быть статическим и публичным.

## Пример задания

Пример класса задания:

```csharp
namespace Module__
{
    internal class Task1
    {
        public static void Execute()
        {
            Console.WriteLine("Задание 1 выполнено.");
        }
    }
}
```
